Class: Atom

A chemical entity consisting of exactly one atomic nucleus and the electron(s) orbiting it.

URI: chemschema:Atom

Parents

  • is_a: ChemicalEntity - An entity that can be described using using chemical properties.

Children

  • AtomIonicForm - an atom type in which charge state is specified. Example: Chloride is an atom ionic species; its formula is Cl−.
  • ChemicalElement - generic form of an atom, with unspecified neutron or charge
  • FullySpecifiedAtom - An atom type that has atomic number, charge, and neutron number stated
  • Nuclide - an atomic species characterized by the specific constitution of its nucleus
